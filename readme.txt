== Description ==

This plugin allows the users (with the ability to edit ‘Orders’) to calculate product pricing excluding tax.

This is especially handy for custom orders that may require tax to be inclusive – something WooCommerce does not support with pricing in their ‘Orders’.

A few notes about the sections above:

== Frequently Asked Questions ==

= Where will this plugin be located? =

This plugin adds a new metabox to your "Orders" page of WooCommerce.
